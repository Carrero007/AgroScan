O Problema

Cada dia sem diagnóstico custa colheita

Agricultores familiares perdem até 40% da produção por não conseguir identificar pragas e doenças a tempo. O acesso a engenheiros agrônomos é caro, lento e geograficamente limitado. A solução precisa estar no bolso — funcionar no campo, offline, em linguagem simples.

Fluxo do Aplicativo

Simples como tirar uma foto

Quatro passos para o agricultor obter o diagnóstico e a solução — sem precisar de internet.

📸
1. Fotografar
O produtor aponta a câmera para a folha, caule ou fruto afetado e tira uma foto.
🧠
2. Identificar
O modelo de visão computacional (embarcado no dispositivo) analisa a imagem offline.
🌿
3. Diagnosticar
O app exibe o nome da praga ou doença com ícone e linguagem acessível, sem jargão técnico.
💊
4. Tratar
Sugestões de tratamento agroecológico e de baixo custo, com ingredientes de fácil acesso.

Funcionalidades

Feito para quem trabalha no campo

📶
100% Offline
O modelo de IA roda diretamente no celular, sem precisar de internet. Funciona em zonas rurais sem sinal.
💬
Linguagem Simples
Interface com ícones e textos curtos. Nenhum termo técnico sem explicação. Acessível para qualquer nível de escolaridade.
🌱
Tratamentos Agroecológicos
Receitas com ingredientes caseiros e de baixo custo. Prioridade para soluções sem agrotóxicos sintéticos.
🔬
Visão Computacional
Modelo treinado com imagens reais de plantações brasileiras para alta precisão no diagnóstico.
📚
Base de Conhecimento Local
Pragas e doenças mapeadas para as culturas mais comuns da agricultura familiar brasileira.
📋
Histórico de Diagnósticos
O produtor pode registrar e acompanhar o histórico de saúde de suas plantas ao longo das safras.

📖 Sobre o TCC

Projeto Acadêmico

O que este trabalho aborda

  • Desenvolvimento de modelo de visão computacional para classificação de doenças em plantas
  • Criação de dataset com imagens de culturas da agricultura familiar brasileira
  • Design de interface acessível e inclusiva para usuários com baixa escolaridade digital
  • Implementação offline com modelo embarcado (TensorFlow Lite / ONNX)
  • Curadoria de tratamentos agroecológicos validados agronomicamente
  • Testes de usabilidade com agricultores familiares reais
⚙️ Tecnologias

Stack & Ferramentas

Python TensorFlow React Native TFLite Transfer Learning PlantVillage Dataset EfficientNet SQLite Expo UX Research

O projeto combina inteligência artificial embarcada, design centrado no usuário e conhecimento agroecológico para democratizar o acesso a diagnósticos fitossanitários no Brasil.
